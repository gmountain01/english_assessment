3단계 AI 채점 파이프라인

Multi-trait 분리 채점: 한 번에 모든 것을 묻지 않고 단계별로 분리하여 성능을 극대화합니다.

Step 1 내용 채점
CoT(생각의 사슬) + Few-Shot 예시로 핵심 명제별 점수 부여
→ 중간 값 없이 각 항목은 0점 또는 만점만 부여 (중간 점수 불허)
↓ content_scores 전달
Step 2 감점 + 피드백
Shute 형성적 피드백 프레임워크 기반
→ 문법/스펠링/어휘/충실도 감점 + Hattie & Timperley 3수준 피드백
↓ full_result 전달
Step 3 자체 검증 (Self-Refine)
수식 재검증 · 과잉칭찬/과소평가 탐지 · 피드백-답안 일치 확인
↓ verified_result
Code applyScoringFormula() 강제 재계산
AI 산술 오류 방지 · 문법 배율 강제 적용 · 영역별 cap 클램핑 → 최종 점수 확정
핵심 수식
FinalScore = Σ max(0, Ci − fdi − (gi + si + vi))
C=내용 점수, fd=충실도 감점, g/s/v=문법·스펠링·어휘 감점. 요소 단위로 0 하한 적용 후 합산.
gramMult = 0.5 + 0.25 × (g − 1)
문법 민감도 슬라이더(g=1~5) → 감점 배율(0.5x~1.5x). 코드에서 강제 적용.
implicitCredit = [1.0, 0.75, 0.5, 0.25, 0.0]
내용 엄격도(1~5) → 인정 판단 기준 조절. 각 항목은 0점 또는 만점만 부여.
κ = (Po − Pe) / (1 − Pe)
가중 Cohen's Kappa. AI-사람 채점의 우연 보정 일치도.
P(θ) = ea(θ−b) / (1 + ea(θ−b))
IRT 2PL 문항반응함수. θ=능력, a=변별도, b=난이도.
BKT: p+ = p·(1−S) / [p·(1−S) + (1−p)·G]   →   pnext = p+ + (1−p+)·T
베이지안 지식추적. S=Slip, G=Guess, T=Transition. 정답 관찰 시 숙달 확률 갱신 후 학습 전이 반영.

오류 감점 규칙

내용 요소별 범위 안에서 발생한 오류를 영역별로 차감합니다. 영역별 최대 감점을 설정하세요.

최종 점수 산출식 (설계 보고서 4.3)
FinalScore = Σ  max( 0,  Content_Elementᵢ − faithfulness_deductᵢ − (문법+스펠링+어휘 감점 합)ᵢ )
충실도(faithfulness) 감점과 영역별 감점을 요소 단위로 먼저 적용하고 0 하한을 건 뒤 합산합니다. 프로그램이 이 공식대로 점수를 강제 재계산합니다.

BKT 시뮬레이션 — 영역별 숙달 추적

베이지안 지식추적(Bayesian Knowledge Tracing)으로 학생의 영역별 숙달 확률이 5회차 평가에 걸쳐 어떻게 변화하는지 시뮬레이션합니다. 채점 결과가 있으면 실제 감점 데이터를 1회차에 반영합니다.

IRT 시뮬레이션 — 잠재 능력 추정

문항반응이론(IRT) 2모수 모형으로 학생의 잠재 요약 능력(θ)을 추정합니다. 5개의 가상 지문(난이도·변별도 상이)에 대한 정오답을 시뮬레이션합니다.

가상 지문 5개의 모수: 쉬운 지문(b=-1.0) ~ 어려운 지문(b=1.5). 변별도(a)는 0.8~1.5. 채점 결과가 있으면 학생별 총점을 기반으로 정오답 패턴을 생성합니다.
